Projeto

Pokédex.

Este projeto foi desenvolvido para fins de estudo da biblioteca React JS com o consumo de uma API REST (PokeAPI).

  • Link do site
  • Link do código no GitHub

Informações

  • Styled Components

    Biblioteca que utiliza o conceito de CSS-in-JS, ou seja, que nos permite escrever códigos CSS dentro do JS.

  • Axios

    Cliente HTTP baseado em Promises para fazer requisições.

  • React Router Dom

    Biblioteca para que você consiga fazer o roteamento das páginas da sua aplicação de forma dinâmica.

  • ReactJs

    Biblioteca JavaScript para a criação de interfaces de usuário ou UI (user interface).

Funcionalidades

  • Listagem dos pokémons
  • Botão que amplia a quantidade de pokémons
  • Método para filtrar os pokémons a partir do seu nome ou ID
  • Responsivo
  • NavTabs para navegar entre as seções
